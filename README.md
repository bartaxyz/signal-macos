# Signal macOS

A native macOS Signal client built with SwiftUI.

## Architecture

- **Frontend:** SwiftUI (macOS 14+)
- **Backend:** signal-cli (handles Signal Protocol, device linking, message transport)
- **Goal:** Replace the Electron-based Signal Desktop with a lightweight native app
